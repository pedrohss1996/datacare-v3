# DataCare

Sistema de Gestão Hospitalar Modular desenvolvido com arquitetura Monólito Modular, integrando bancos de dados MariaDB e Oracle (Legado).

## 🚀 Tecnologias

- **Backend:** Node.js, Express
- **Frontend:** EJS, TailwindCSS
- **Database:** MariaDB (Knex.js) + Oracle (Integration)
- **AI Integration:** Google Gemini (Page Builder & Voice Commands)
